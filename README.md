# Enchanting Rework
![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

A comprehensive overhaul of Minecraft's enchanting mechanics, designed as a server-side only mod! Enchanting Rework completely changes how players interact with the Enchanting Table, Anvils, and Grindstones.

## Current Features
*   Enchanting items (tools, weapons, armor) at the Enchanting Table costs 0 experience levels! (Note: enchanting Books still functions normally).
*   **"Free" Tag Tracking**: Items enchanted for free are silently tagged with a "free" status. This tag persists when combining or repairing items in an Anvil.
*   **Anvil Repair Rework**: Repairing items has a flat cost of 1 level, effectively removing the dreaded "Too Expensive!" restriction and keeping repairs predictable.
*   **Grindstone Disenchanting Drawbacks**: Removing enchantments from "free" items in a Grindstone grants no experience back and actually damages the item based on the removed enchantments' value!
*   **Grindstone Book Splitting**: Got an enchanted book with multiple enchantments? Place it in the top slot of a Grindstone and regular books in the bottom slot to split the last enchantment off into a new book!
*   **Tag Removal**: Disenchanting an item successfully removes its "free" tag, returning it to standard vanilla behavior.
*   **Server-Side Only**: Can be installed purely on the server. Clients do not need to download the mod!
